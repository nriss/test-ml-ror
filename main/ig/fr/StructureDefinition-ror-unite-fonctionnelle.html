Modèle logique: ROR - Unité Fonctionnelle

URL officielle: https://interop.esante.gouv.fr/ig/fhir/test-ror/StructureDefinition/ror-unite-fonctionnelle Version: 0.1.0
Draft à partir de 2026-06-09 Nom computable: RORUniteFonctionnelle

Spécialisation ROR de l'UniteFonctionnelle du MOS. Plus petite unité compatible avec les contraintes de gestion présentant une activité médicale homogène.

Utilisations:

  • Ce Modèle logique n'est utilisé par aucun autre profil dans ce guide d'implémentation

Vous pouvez également vérifier les usages dans le FHIR IG Statistics
