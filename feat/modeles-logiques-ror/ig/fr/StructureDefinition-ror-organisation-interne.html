Modèle logique: ROR - Organisation Interne ( Expérimental )

URL officielle: https://interop.esante.gouv.fr/ig/fhir/test-ror/StructureDefinition/ror-organisation-interne Version: 0.1.0
Draft à partir de 2026-06-01 Nom computable: RorOrganisationInterne

Modèle logique ROR représentant une organisation interne d'une structure de santé (pôle, service, unité fonctionnelle). Cette entité n'existe pas dans le MOS et est propre au ROR.

Utilisations:

  • Référence ce Modèle logique: ROR - Entité Géographique and ROR - Offre Opérationnelle

Vous pouvez également vérifier les usages dans le FHIR IG Statistics
