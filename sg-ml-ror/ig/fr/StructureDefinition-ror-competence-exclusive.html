Modèle logique: ROR - Compétence Exclusive

URL officielle: https://interop.esante.gouv.fr/ig/fhir/test-ror/StructureDefinition/ror-competence-exclusive Version: 0.1.0
Draft à partir de 2026-07-07 Nom computable: RORCompetenceExclusive

Spécialisation ROR de la CompetenceExclusive du MOS. Compétence exercée à titre exclusif, incompatible avec l'exercice d'une spécialité.

Utilisations:

  • Ce Modèle logique n'est utilisé par aucun autre profil dans ce guide d'implémentation

Vous pouvez également vérifier les usages dans le FHIR IG Statistics
